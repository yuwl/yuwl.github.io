解释

  1. public class HelloWorld: 这是定义一个名为 HelloWorld 的公共类。在 Java 中，每个程序都必须有一个公共类，并且类名必须与文件名相同（包括大小写）。

  2. public static void main(String[] args): 这是程序的入口点，也就是程序开始执行的地方。

  • public: 表示该方法是公开的，可以从其他类访问。
  • static: 表示该方法是静态的，可以不创建对象直接调用。
  • void: 表示该方法没有返回值。
  • main: 是 Java 虚拟机（JVM）执行程序时默认调用的方法。
  • String[] args: 表示 main 方法可以接收一个字符串数组作为参数，这些参数可以用来从命令行传递数据到程序。
  1. System.out.println(“Hello, World!”);: 这行代码的作用是将字符串 “Hello, World!” 打印到控制台。
  • System: 是一个包含系统信息的类，其中 out 是一个标准输出流对象，println() 是一个用于输出字符串的方法。

如何运行程序

  1. 将代码保存为 HelloWorld.java 文件。
  2. 打开命令提示符或终端，进入保存文件所在的目录。
  3. 使用 javac HelloWorld.java 命令编译代码。
  4. 使用 java HelloWorld 命令运行程序，你将在控制台看到 “Hello, World!” 的输出。
