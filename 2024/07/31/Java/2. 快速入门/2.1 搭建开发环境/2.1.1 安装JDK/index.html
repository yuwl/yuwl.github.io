2.1.1 安装JDK

Yuwl 2024-07-31 11:06:06
Categories： Java Tags： java

在 Windows、Linux 和 macOS 上安装 JDK 8 的步骤：

Windows

  1. 下载 JDK 8： 访问 Oracle 的官方网站
    https://www.oracle.com/java/technologies/javase/javase8-archive-downloads.html
    ，找到 JDK 8 的下载页面，选择 Windows 平台的安装程序，例如 jdk-8u341-windows-x64.exe。

  2. 运行安装程序： 双击下载的安装程序，按照安装向导的提示进行操作。建议接受默认的安装路径，例如 C:\Program Files\Java\jdk1.8.0_341。

  3. 配置环境变量：

  • 右键点击 “此电脑” -> “属性” -> “高级系统设置” -> “环境变量”。
  • 在 “系统变量” 中，找到名为 “PATH” 的变量，点击 “编辑”。
  • 在 “变量值” 中，添加 JDK 8 的安装路径，例如 C:\Program Files\Java\jdk1.8.0_341\bin，并用分号与其他路径分隔。
  • 另外，新建一个系统变量，名为 JAVA_HOME，变量值为 JDK 8 的安装路径，例如 C:\Program Files\Java\jdk1.8.0_341。
  1. 验证安装： 打开命令提示符，输入 java -version，如果成功安装，则会显示 JDK 8 的版本信息。

Linux

    https://www.oracle.com/java/technologies/javase/javase8-archive-downloads.html
