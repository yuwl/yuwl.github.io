8.2 List

Categories： Java Tags： java

现在我们来学习 Java 中的集合 List。

List是一个有序的集合，允许重复元素。Java中有几个常用的List实现类：ArrayList、LinkedList和Vector。我们将重点介绍ArrayList和LinkedList。

ArrayList

ArrayList是基于动态数组实现的List，它提供了快速的随机访问性能。它的主要特点是：

  1. 优点: 随机访问快，增删操作在末尾时较快。
  2. 缺点: 在中间插入或删除元素时较慢，因为需要移动其他元素。

LinkedList

LinkedList是基于双向链表实现的List，它提供了快速的插入和删除操作。它的主要特点是：

  1. 优点: 插入和删除操作快，特别是在列表的开头和中间。
  2. 缺点: 随机访问速度慢，因为需要从头或尾遍历链表。

  • LinkedList 适合频繁插入和删除操作，但随机访问较少的场景。

Vector

1. 什么是Vector类？

Vector类是Java集合框架中的一个类，用于存储动态数组。它类似于ArrayList，但Vector是线程安全的，即它的方法是同步的，因此多个线程可以同时访问它而不会引起数据不一致问题。然而，由于同步的开销，Vector通常比ArrayList稍慢。

5. Vector与ArrayList的区别

虽然Vector和ArrayList都实现了List接口，但它们之间有一些关键的区别：

  • 线程安全：Vector是同步的（线程安全），而ArrayList不是。
  • 性能：由于同步的开销，Vector通常比ArrayList慢。
  • 扩容机制：Vector的默认扩容机制是当前容量的2倍，而ArrayList是当前容量的1.5倍。

Stack

1. 什么是Stack类？

Stack是Java集合框架中的一个类，用于实现栈数据结构。栈是一种后进先出（LIFO, Last In First Out）结构，这意味着最后添加的元素将是第一个被移除的元素。

Stack类继承自Vector类，并提供了几个特定于栈的方法：

  • push(E item): 将元素压入栈顶。
  • pop(): 移除并返回栈顶元素。
  • peek(): 返回栈顶元素但不移除它。
  • isEmpty(): 检查栈是否为空。
  • search(Object o): 返回元素在栈中的位置，以1为基准（从栈顶开始计数）。
