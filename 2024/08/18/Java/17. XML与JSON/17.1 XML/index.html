XML简介

XML是什么？

XML（可扩展标记语言，eXtensible Markup Language）是一种用于存储和传输数据的标记语言。它的设计目的是强调数据的结构和可扩展性，而不是外观。XML既可以由人类阅读，也可以由机器解析，这使得它在跨平台的数据交换中非常有用。

特点：

  • 可扩展性：XML没有固定的标签集合，用户可以根据需要自定义标签。
  • 自描述性：XML数据结构清晰，通过标签名就可以理解数据的含义。
  • 平台无关性：XML是纯文本格式，能够在不同的平台和系统之间无障碍地传输和读取。

  • 适用于小型XML文件，对于大型文档，DOM会占用大量内存。

2. 使用SAX解析器

SAX（Simple API for XML）解析器是基于事件驱动的解析方式，不会将整个XML文档加载到内存中，而是逐行解析，适用于处理大文件或流式数据。

优点：

  • 内存消耗低，适合处理大型XML文件。

  • 解析速度快。
    缺点：

  • 基于事件的回调机制，代码逻辑较复杂。

  • 不能直接访问和修改文档结构。

3. 使用StAX解析器

StAX（Streaming API for XML）是一种基于光标的解析方法，与SAX类似，但提供了更灵活的操作方式，允许前后遍历XML文档，适用于既需要流式处理又需要一定文档操作的场景。
