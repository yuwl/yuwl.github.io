2.2.4 жө®зӮ№ж•°иҝҗз®—

Yuwl 2024-08-01 10:53:26
Categoriesпјҡ Java Tagsпјҡ java

зҺ°еңЁжҲ‘д»¬жқҘеӯҰд№  Java дёӯзҡ„жө®зӮ№ж•°иҝҗз®—гҖӮ

1. жө®зӮ№ж•°зұ»еһӢ

Java жҸҗдҫӣдёӨз§Қжө®зӮ№ж•°зұ»еһӢпјҡ

  • floatпјҡ еҚ•зІҫеәҰжө®зӮ№ж•°пјҢдҪҝз”Ё 32 дҪҚеӯҳеӮЁпјҢзІҫеәҰиҫғдҪҺгҖӮ
  • doubleпјҡ еҸҢзІҫеәҰжө®зӮ№ж•°пјҢдҪҝз”Ё 64 дҪҚеӯҳеӮЁпјҢзІҫеәҰиҫғй«ҳгҖӮdouble жҳҜ Java дёӯй»ҳи®Өзҡ„жө®зӮ№ж•°зұ»еһӢгҖӮ

2. жө®зӮ№ж•°иҝҗз®—з¬Ұ

жө®зӮ№ж•°иҝҗз®—з¬ҰдёҺж•ҙж•°иҝҗз®—з¬Ұеҹәжң¬зӣёеҗҢпјҢеҢ…жӢ¬пјҡ

  • + (еҠ жі•)пјҡе°ҶдёӨдёӘжө®зӮ№ж•°зӣёеҠ гҖӮ
  • - (еҮҸжі•)пјҡе°ҶдёӨдёӘжө®зӮ№ж•°зӣёеҮҸгҖӮ
  • * (д№ҳжі•)пјҡе°ҶдёӨдёӘжө®зӮ№ж•°зӣёд№ҳгҖӮ
  • / (йҷӨжі•)пјҡе°ҶдёӨдёӘжө®зӮ№ж•°зӣёйҷӨгҖӮ
  • % (еҸ–жЁЎ)пјҡи®Ўз®—дёӨдёӘжө®зӮ№ж•°зӣёйҷӨзҡ„дҪҷж•°гҖӮ

3. жө®зӮ№ж•°иҝҗз®—зӨәдҫӢ

4. жө®зӮ№ж•°зІҫеәҰй—®йўҳ

з”ұдәҺи®Ўз®—жңәдҪҝз”ЁдәҢиҝӣеҲ¶иЎЁзӨәжө®зӮ№ж•°пјҢиҖҢеҫҲеӨҡеҚҒиҝӣеҲ¶ж•°ж— жі•зІҫзЎ®ең°з”ЁдәҢиҝӣеҲ¶иЎЁзӨәпјҢеӣ жӯӨжө®зӮ№ж•°иҝҗз®—еҸҜиғҪдјҡеӯҳеңЁзІҫеәҰй—®йўҳгҖӮ

5. дҪҝз”Ё BigDecimal еӨ„зҗҶзІҫеәҰй—®йўҳ

дёәдәҶйҒҝе…Қжө®зӮ№ж•°зІҫеәҰй—®йўҳпјҢеҸҜд»ҘдҪҝз”Ё java.math.BigDecimal зұ»жқҘиҝӣиЎҢзІҫзЎ®зҡ„жө®зӮ№ж•°иҝҗз®—гҖӮ

6. int зұ»еһӢдёҺ float жҲ– double зұ»еһӢд№Ӣй—ҙзҡ„иҪ¬жҚў

1. д»Һ int иҪ¬жҚўдёә float жҲ– doubleпјҡ

  • иҮӘеҠЁзұ»еһӢиҪ¬жҚў (Widening Conversion)пјҡ еҪ“ int дёҺ float жҲ– double еҸӮдёҺиҝҗз®—ж—¶пјҢint дјҡиҮӘеҠЁиҪ¬жҚўдёә float жҲ– doubleгҖӮдҫӢеҰӮпјҡ
  • ејәеҲ¶зұ»еһӢиҪ¬жҚў (Casting)пјҡ дҪ д№ҹеҸҜд»ҘдҪҝз”ЁејәеҲ¶зұ»еһӢиҪ¬жҚўжқҘжҳҫејҸең°е°Ҷ int иҪ¬жҚўдёә float жҲ– doubleгҖӮдҫӢеҰӮпјҡ

2. д»Һ float жҲ– double иҪ¬жҚўдёә intпјҡ

  • ејәеҲ¶зұ»еһӢиҪ¬жҚў (Casting)пјҡ з”ұдәҺ float жҲ– double зҡ„зІҫеәҰжҜ” int й«ҳпјҢзӣҙжҺҘе°Ҷ float жҲ– double иөӢеҖјз»ҷ int дјҡйҖ жҲҗж•°жҚ®дёўеӨұгҖӮеӣ жӯӨпјҢдҪ йңҖиҰҒдҪҝз”ЁејәеҲ¶зұ»еһӢиҪ¬жҚўгҖӮдҫӢеҰӮпјҡ

  • жіЁж„Ҹж•°жҚ®дёўеӨұпјҡ ејәеҲ¶зұ»еһӢиҪ¬жҚўеҸҜиғҪдјҡеҜјиҮҙж•°жҚ®дёўеӨұпјҢеӣ дёә float жҲ– double дёӯзҡ„е°Ҹж•°йғЁеҲҶдјҡиў«иҲҚејғгҖӮ

жҖ»з»“:

  • Java жҸҗдҫӣдәҶдёӨз§Қжө®зӮ№ж•°зұ»еһӢпјҡfloat е’Ң doubleгҖӮ
  • жө®зӮ№ж•°иҝҗз®—з¬ҰдёҺж•ҙж•°иҝҗз®—з¬Ұзұ»дјјгҖӮ
  • жө®зӮ№ж•°иҝҗз®—еҸҜиғҪдјҡеӯҳеңЁзІҫеәҰй—®йўҳпјҢеҸҜд»ҘдҪҝз”Ё BigDecimal зұ»жқҘиҝӣиЎҢзІҫзЎ®зҡ„жө®зӮ№ж•°иҝҗз®—гҖӮ
  • int зұ»еһӢдёҺ float жҲ– double зұ»еһӢд№Ӣй—ҙзҡ„иҪ¬жҚў

еёҢжңӣиҝҷдәӣдҝЎжҒҜиғҪеӨҹеё®еҠ©дҪ зҗҶи§Ј Java дёӯзҡ„жө®зӮ№ж•°иҝҗз®—пјҒ
