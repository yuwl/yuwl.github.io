2.2.4 浮点数运算

Yuwl 2024-08-01 10:53:26
Categories： Java Tags： java

现在我们来学习 Java 中的浮点数运算。

1. 浮点数类型

Java 提供两种浮点数类型：

  • float： 单精度浮点数，使用 32 位存储，精度较低。
  • double： 双精度浮点数，使用 64 位存储，精度较高。double 是 Java 中默认的浮点数类型。

2. 浮点数运算符

浮点数运算符与整数运算符基本相同，包括：

  • + (加法)：将两个浮点数相加。
  • - (减法)：将两个浮点数相减。
  • * (乘法)：将两个浮点数相乘。
  • / (除法)：将两个浮点数相除。
  • % (取模)：计算两个浮点数相除的余数。

4. 浮点数精度问题

由于计算机使用二进制表示浮点数，而很多十进制数无法精确地用二进制表示，因此浮点数运算可能会存在精度问题。

5. 使用 BigDecimal 处理精度问题

为了避免浮点数精度问题，可以使用 java.math.BigDecimal 类来进行精确的浮点数运算。
