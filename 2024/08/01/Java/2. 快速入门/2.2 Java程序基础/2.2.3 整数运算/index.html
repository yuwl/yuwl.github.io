2.2.3 整数运算

Yuwl 2024-08-01 10:35:40
Categories： Java Tags： java

接下来我们来学习 Java 中的整数运算。

1. 整数类型

Java 提供了四种整数类型，分别是：

  • byte： 8 位有符号整数，取值范围为 -128 到 127。
  • short： 16 位有符号整数，取值范围为 -32768 到 32767。
  • int： 32 位有符号整数，取值范围为 -2147483648 到 2147483647。这是最常用的整数类型。
  • long： 64 位有符号整数，取值范围非常大，可以表示更大的数字。如果需要存储非常大的整数，可以使用 long 类型。

2. 整数运算符

Java 提供了以下整数运算符：

  • + (加法)：将两个整数相加。
  • - (减法)：将两个整数相减。
  • * (乘法)：将两个整数相乘。
  • / (除法)：将两个整数相除，结果为整数，余数会被舍弃。
  • % (取模)：将两个整数相除，返回余数。

4. 运算符优先级

运算符优先级决定了表达式中运算符的执行顺序。Java 中的运算符优先级与数学中的优先级相同。

  • 乘法、除法、取模 优先于 加法、减法。
  • 括号 可以改变运算符的优先级。

5. 整数溢出

如果整数运算的结果超出了该类型所能表示的范围，就会发生整数溢出。例如，int 类型最大值为 2147483647，如果两个 int 类型变量相加的结果超过了这个值，就会发生溢出。

6. 整数类型转换

不同整数类型之间运算时，会发生类型转换，称为自动类型提升 (Automatic Type Promotion)。

自动类型提升：

  • 参与运算的类型：运算中会将所有参与运算的类型提升到最高类型。
  • 最高类型：不同类型的整数参与运算时，会提升到其中最大的类型。例如：byte、short、char 参与运算时，会提升到 int 类型。
  • 提升规则: 如果参与运算的类型中包含 long，则结果类型为 long；否则结果类型为 int。

显式类型转换:

如果需要将结果类型强制转换为其他类型，可以使用强制类型转换 (Casting)，但需要注意可能造成数据丢失。

总结:

  • Java 提供了四种整数类型，分别是 byte、short、int 和 long。
  • Java 提供了常见的整数运算符，例如加法、减法、乘法、除法和取模。
  • 运算符优先级决定了表达式中运算符的执行顺序。
  • 如果整数运算的结果超出了该类型所能表示的范围，就会发生整数溢出。
  • 可以使用类型转换运算符将一个整数类型转换为另一个整数类型。

希望这些信息能够帮助你理解 Java 中的整数运算！
