2. 数据类型

数据类型决定了变量可以存储什么类型的数据。Java 中的数据类型分为两类：

  • 基本数据类型 (Primitive Data Types)： 这些类型是 Java 内置的，可以直接使用。
  • 引用数据类型 (Reference Data Types)： 这些类型是通过类或接口定义的，存储的是对象的引用。

基本数据类型：

数据类型 描述 示例
byte 8 位有符号整数 byte b = 10;
short 16 位有符号整数 short s = 200;
int 32 位有符号整数 int i = 1000;
long 64 位有符号整数 long l = 10000000000L;
float 32 位单精度浮点数 float f = 3.14f;
double 64 位双精度浮点数 double d = 3.141592653589793;
char 16 位 Unicode 字符 char c = ‘A’;
boolean 布尔值 (true 或 false) boolean b = true;

引用数据类型：

  • 类：例如 String、ArrayList 等。
  • 接口：例如 List、Map 等。
  • 数组：例如 int[] numbers 等。
