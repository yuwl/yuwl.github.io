2.3.2 if条件判断

Yuwl 2024-08-01 19:51:07
Categories： Java Tags： java

现在我们来学习 Java 中的 if 条件判断。

if 条件判断 是程序中控制流程的关键工具，它允许程序根据不同的条件执行不同的代码块。

基本语法：

解释：

  • 条件表达式: 这是一个布尔表达式，它可以评估为 true 或 false。
  • 代码块: 这是一组用花括号 {} 包裹的语句，它们将在满足特定条件时执行。

示例：

在这个示例中：

  • age >= 18 是条件表达式。
  • 如果 age 大于或等于 18，程序将输出 “您已成年”。
  • 如果 age 小于 18，程序将输出 “您未成年”。

if 条件判断的其他形式：

  • if-else if-else: 用于判断多个条件。

  • 嵌套 if: 一个 if 语句嵌套在另一个 if 语句中。
