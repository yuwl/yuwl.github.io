2.3.4 while循环

Yuwl 2024-08-01 20:13:50
Categories： Java Tags： java

现在我们来学习 Java 中的 while 循环。

while 循环是一种重复执行代码块的控制流语句，它会一直执行代码块，直到循环条件变为 false。

语法:

工作原理:

  1. 首先，程序会评估 条件表达式。
  2. 如果 条件表达式 为 true，则执行 代码块 中的代码。
  3. 完成 代码块 的执行后，程序会再次评估 条件表达式。
  4. 重复步骤 2 和 3，直到 条件表达式 为 false，此时循环结束。

要点:

  • while 循环的条件表达式必须能够计算为 boolean 类型。
  • 如果条件表达式在开始时就为 false，那么循环体将不会被执行。
  • 为了防止无限循环，循环体中必须包含能够改变条件表达式值的语句，例如，在上面的示例中，i++ 语句改变了计数器 i 的值。

使用场景:

  • 当你不知道循环次数时，使用 while 循环非常适合。
  • 当你想要重复执行代码块，直到满足某个条件时，可以使用 while 循环。

注意事项:

  • 确保循环条件最终会变为 false，否则程序会陷入无限循环。
  • 在循环中使用变量时，要确保变量的初始值和更新方式是正确的，以避免循环无法正常结束。

break 语句

break 语句用于立即退出当前 while 循环，程序控制流会跳到循环之后的下一条语句。通常用于满足特定条件时提前结束循环，避免不必要的迭代。

continue 语句

continue 语句用于跳过当前循环迭代中剩余的代码，并立即开始下一个循环迭代。通常用于遇到特定条件时，跳过当前迭代，继续执行之后的迭代。
