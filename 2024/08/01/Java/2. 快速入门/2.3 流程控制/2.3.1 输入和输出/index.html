2.3.1 输入和输出

Yuwl 2024-08-01 19:26:49
Categories： Java Tags： java

现在我们来学习 Java 中的输入和输出。

1. 输入 (Input):

  • Java 使用 Scanner 类来获取用户从键盘输入的数据。
  • 首先，你需要创建一个 Scanner 对象，并将其与标准输入流 System.in 关联。
  • 然后，你可以使用 Scanner 类的各种方法来读取不同类型的数据，例如：
    • nextInt()：读取整数
    • nextDouble()：读取浮点数
    • nextLine()：读取一行文本

2. 输出 (Output):

  • Java 使用 System.out 对象来向控制台输出数据。
  • 使用 System.out.println() 方法可以打印数据并换行。
  • 使用 System.out.print() 方法可以打印数据，但不换行。
