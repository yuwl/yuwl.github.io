2.3.7 break和continue

Yuwl 2024-08-02 13:22:47
Categories： Java Tags： java

现在我们来学习 Java 中的 break 和 continue 。

break 和 continue 是 Java 中的两种控制流语句，用于在循环中改变程序的执行流程。

1. break 语句：

用途：break 语句用于立即退出当前循环，并将程序控制转移到循环之后的下一条语句。

适用场景：

  • 当满足特定条件时，需要提前结束循环。
  • 当循环中出现错误或异常时，需要立即停止循环。

示例：

这个示例程序会打印从 1 到 4 的数字，当 i 等于 5 时，break 语句会使循环提前结束。

2. continue 语句：

用途：continue 语句用于跳过当前循环迭代中剩余的代码，并立即开始下一个循环迭代。

适用场景：

  • 当遇到特定条件时，需要跳过当前迭代，继续执行之后的迭代。
  • 当需要在循环中执行某些特定的操作时，但不希望影响其他迭代。

示例：

这个示例程序会打印从 1 到 10 的奇数。当 i 是偶数时，continue 语句会跳过循环，继续执行下一个循环迭代。

注意：break 和 continue 语句只能在循环中使用，不能在其他语句块中使用。

要点：

  • break 语句用于立即退出循环，并将程序控制转移到循环之后的下一条语句。
  • continue 语句用于跳过当前循环迭代中剩余的代码，并立即开始下一个循环迭代。
  • break 和 continue 语句通常用于在循环中改变程序的执行流程，但它们也可以用于在其他语句块中使用。
