3.1.7 йқҷжҖҒеӯ—ж®өе’ҢйқҷжҖҒж–№жі•

Yuwl 2024-08-03 10:31:13
Categoriesпјҡ Java Tagsпјҡ java

зҺ°еңЁжҲ‘д»¬жқҘеӯҰд№  Java йқўеҗ‘еҜ№иұЎдёӯзҡ„йқҷжҖҒеӯ—ж®өе’ҢйқҷжҖҒж–№жі•гҖӮ

д»Җд№ҲжҳҜйқҷжҖҒеӯ—ж®өе’ҢйқҷжҖҒж–№жі•пјҹ

  1. йқҷжҖҒеӯ—ж®өпјҲStatic Fieldsпјүпјҡд№ҹз§°дёәзұ»еҸҳйҮҸпјҢеұһдәҺзұ»иҖҢдёҚжҳҜзұ»зҡ„е®һдҫӢгҖӮ
  2. йқҷжҖҒж–№жі•пјҲStatic MethodsпјүпјҡеұһдәҺзұ»иҖҢдёҚжҳҜзұ»зҡ„е®һдҫӢпјҢеҸҜд»ҘзӣҙжҺҘйҖҡиҝҮзұ»еҗҚи°ғз”ЁгҖӮ

йқҷжҖҒеӯ—ж®ө

зү№зӮ№

  1. дҪҝз”Ё static е…ій”®еӯ—еЈ°жҳҺгҖӮ
  2. жүҖжңүе®һдҫӢе…ұдә«еҗҢдёҖдёӘйқҷжҖҒеӯ—ж®өгҖӮ
  3. еҸҜд»ҘйҖҡиҝҮзұ»еҗҚзӣҙжҺҘи®ҝй—®пјҢж— йңҖеҲӣе»әзұ»зҡ„е®һдҫӢгҖӮ
  4. йҖҡеёёз”ЁдәҺиЎЁзӨәжүҖжңүе®һдҫӢе…ұжңүзҡ„еұһжҖ§гҖӮ

зӨәдҫӢ

йқҷжҖҒж–№жі•

зү№зӮ№

  1. дҪҝз”Ё static е…ій”®еӯ—еЈ°жҳҺгҖӮ
  2. еҸҜд»ҘзӣҙжҺҘйҖҡиҝҮзұ»еҗҚи°ғз”ЁпјҢж— йңҖеҲӣе»әзұ»зҡ„е®һдҫӢгҖӮ
  3. еҸӘиғҪзӣҙжҺҘи®ҝй—®йқҷжҖҒжҲҗе‘ҳпјҲйқҷжҖҒеӯ—ж®өе’ҢйқҷжҖҒж–№жі•пјүгҖӮ
  4. дёҚиғҪдҪҝз”Ё this е…ій”®еӯ—гҖӮ

зӨәдҫӢ

йқҷжҖҒеҲқе§ӢеҢ–еқ—

йқҷжҖҒеҲқе§ӢеҢ–еқ—з”ЁдәҺеҲқе§ӢеҢ–йқҷжҖҒеӯ—ж®өпјҢе®ғеңЁзұ»еҠ иҪҪж—¶жү§иЎҢгҖӮ

йқҷжҖҒеҜје…Ҙ

йқҷжҖҒеҜје…Ҙе…Ғи®ёдҪ зӣҙжҺҘдҪҝз”ЁйқҷжҖҒжҲҗе‘ҳпјҢиҖҢдёҚйңҖиҰҒзұ»еҗҚйҷҗе®ҡгҖӮ

дҪҝз”ЁеңәжҷҜ

  1. е·Ҙе…·зұ»пјҡеҰӮ Math зұ»пјҢеҢ…еҗ«и®ёеӨҡйқҷжҖҒж–№жі•гҖӮ
  2. еёёйҮҸпјҡеҰӮ Math.PIгҖӮ
  3. е·ҘеҺӮж–№жі•пјҡз”ЁдәҺеҲӣе»әеҜ№иұЎзҡ„йқҷжҖҒж–№жі•гҖӮ
  4. еҚ•дҫӢжЁЎејҸпјҡдҪҝз”ЁйқҷжҖҒеӯ—ж®өеӯҳеӮЁе”ҜдёҖе®һдҫӢгҖӮ

жіЁж„ҸдәӢйЎ№

  1. йқҷжҖҒж–№жі•дёҚиғҪи®ҝй—®йқһйқҷжҖҒжҲҗе‘ҳгҖӮ
  2. йқҷжҖҒж–№жі•дёҚиғҪиў«йҮҚеҶҷпјҢдҪҶеҸҜд»Ҙиў«йҡҗи—ҸгҖӮ
  3. иҝҮеәҰдҪҝз”ЁйқҷжҖҒеҸҜиғҪеҜјиҮҙд»Јз Ғйҡҫд»ҘжөӢиҜ•е’Ңз»ҙжҠӨгҖӮ
