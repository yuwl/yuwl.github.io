4.6 使用SLF4J

Yuwl 2024-08-03 16:46:31
Categories： Java Tags： java

现在我们来学习 Java 中使用 SLF4J。

SLF4J是一个通用的日志记录API，它本身不提供具体的日志记录实现，而是通过绑定（Binding）与具体的日志记录框架（如Log4j）集成。

SLF4J 和 Log4J

Log4j

Log4j是一个功能强大的日志记录框架，可以记录日志到控制台、文件等多种目标。
