4.2 使用断言

现在我们来学习 Java 中的断言。

什么是断言？

断言是一个带有布尔表达式的语句，用于检查程序中的假设。如果布尔表达式为 true，程序继续执行。如果为 false，程序会抛出 AssertionError，并终止执行。

使用断言的最佳实践

  1. 仅用于开发和测试： 断言主要用于开发和测试阶段，不应该依赖它们进行正常的程序逻辑控制。

  2. 避免副作用： 断言不应该改变程序状态或有副作用。例如，不要在断言中调用改变程序状态的方法。

  3. 明确错误信息： 使用带有错误消息的断言可以帮助你更容易地理解断言失败的原因。
