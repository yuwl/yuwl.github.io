4.3 使用JDK Logging

Yuwl 2024-08-03 16:45:44
Categories： Java Tags： java

现在我们来学习 Java 中的 Logging。

JDK Logging是Java内置的日志记录框架，位于 java.util.logging 包中。它可以帮助你记录程序的运行信息、调试信息、错误信息等。

基本概念

Logger

Logger 类是日志记录的核心类，用于记录日志信息。每个 Logger 实例都有一个名称，并且与一个或多个日志处理器（Handler）和日志级别（Level）相关联。

Log Levels

日志级别用于指定日志的严重程度。常用的日志级别有：

  • SEVERE：表示严重错误
  • WARNING：表示潜在问题
  • INFO：表示一般信息
  • CONFIG：表示配置消息
  • FINE：表示详细信息
  • FINER：表示更详细的信息
  • FINEST：表示最详细的信息

Handler

Handler 类用于定义日志的输出位置，例如控制台、文件等。常用的 Handler 有 ConsoleHandler 和 FileHandler。

Formatter

Formatter 类用于定义日志的输出格式。常用的格式化类有 SimpleFormatter 和 XMLFormatter。
