3.1.3 构造方法

Yuwl 2024-08-03 10:29:46
Categories： Java Tags： java

现在我们来学习 Java 面向对象中的构造方法。

什么是构造方法？

构造方法是一种特殊的方法，用于初始化对象。当使用 new 关键字创建对象时，构造方法会被自动调用。

构造方法的特点

  • 构造方法的名称必须与类名完全相同。
  • 构造方法没有返回类型，甚至不能用 void。
  • 构造方法可以有参数，也可以没有参数。
  • 一个类可以有多个构造方法（方法重载）。

默认构造方法

如果你没有为类定义任何构造方法，Java会自动提供一个无参数的默认构造方法。

自定义构造方法

你可以创建自己的构造方法来初始化对象的属性。

使用构造方法创建对象

构造方法重载

你可以为一个类定义多个构造方法，这称为构造方法重载。

this() 调用其他构造方法

在一个构造方法中，你可以使用 this() 来调用同一个类的其他构造方法。

注意事项

  • 如果你定义了任何构造方法，Java 就不会再提供默认构造方法。
  • 构造方法不能被继承，但可以在子类中使用 super() 调用父类的构造方法。
  • 构造方法不能被直接调用，只能在创建对象时通过 new 关键字间接调用。
