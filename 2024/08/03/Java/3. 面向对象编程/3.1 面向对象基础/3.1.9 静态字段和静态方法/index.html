3.1.9 静态字段和静态方法

Yuwl 2024-08-03 10:31:13
Categories： Java Tags： java

现在我们来学习 Java 面向对象中的静态字段和静态方法。

什么是静态字段和静态方法？

  1. 静态字段（Static Fields）：也称为类变量，属于类而不是类的实例。
  2. 静态方法（Static Methods）：属于类而不是类的实例，可以直接通过类名调用。

静态字段

特点

  1. 使用 static 关键字声明。
  2. 所有实例共享同一个静态字段。
  3. 可以通过类名直接访问，无需创建类的实例。
  4. 通常用于表示所有实例共有的属性。

静态方法

特点

  1. 使用 static 关键字声明。
  2. 可以直接通过类名调用，无需创建类的实例。
  3. 只能直接访问静态成员（静态字段和静态方法）。
  4. 不能使用 this 关键字。

使用场景

  1. 工具类：如 Math 类，包含许多静态方法。
  2. 常量：如 Math.PI。
  3. 工厂方法：用于创建对象的静态方法。
  4. 单例模式：使用静态字段存储唯一实例。

注意事项

  1. 静态方法不能访问非静态成员。
  2. 静态方法不能被重写，但可以被隐藏。
  3. 过度使用静态可能导致代码难以测试和维护。
