3.1.2 方法

Yuwl 2024-08-03 10:29:27
Categories： Java Tags： java

现在我们来学习 Java 面向对象中的方法。

什么是方法？

方法是执行特定任务的一段代码。它们可以接收输入参数，执行操作，并返回结果。方法在Java中也被称为函数。

方法的定义

方法的基本结构

一个方法的基本结构包括：

  1. 方法的修饰符（如public、private等）
  2. 返回类型（如int、void等）
  3. 方法名
  4. 参数列表（括号中的变量）
  5. 方法体（大括号中的代码）

带参数的方法

方法可以接受参数，这些参数在方法调用时传递。

带返回值的方法

方法可以返回一个值，返回类型决定了方法返回值的类型。

方法的调用

定义方法后，你可以在类的实例上调用它们。
