3.1.2 方法

Yuwl 2024-08-03 10:29:27
Categories： Java Tags： java

现在我们来学习 Java 面向对象中的方法。

Java 中的方法

什么是方法？

方法是执行特定任务的一段代码。它们可以接收输入参数，执行操作，并返回结果。方法在Java中也被称为函数。

方法的定义

方法的基本结构

一个方法的基本结构包括：

  1. 方法的修饰符（如public、private等）
  2. 返回类型（如int、void等）
  3. 方法名
  4. 参数列表（括号中的变量）
  5. 方法体（大括号中的代码）

带参数的方法

方法可以接受参数，这些参数在方法调用时传递。

带返回值的方法

方法可以返回一个值，返回类型决定了方法返回值的类型。

方法的调用

定义方法后，你可以在类的实例上调用它们。

构造方法

什么是构造方法？

构造方法是一种特殊的方法，用于初始化对象。当使用 new 关键字创建对象时，构造方法会被自动调用。

构造方法的特点

  • 构造方法的名称必须与类名完全相同。
  • 构造方法没有返回类型，甚至不能用 void。
  • 构造方法可以有参数，也可以没有参数。
  • 一个类可以有多个构造方法（方法重载）。

默认构造方法

如果你没有为类定义任何构造方法，Java会自动提供一个无参数的默认构造方法。

自定义构造方法

你可以创建自己的构造方法来初始化对象的属性。

使用构造方法创建对象

构造方法重载

你可以为一个类定义多个构造方法，这称为构造方法重载。

this() 调用其他构造方法

在一个构造方法中，你可以使用 this() 来调用同一个类的其他构造方法。

方法重载

什么是方法重载？

方法重载（Method Overloading）指在同一个类中定义多个方法，这些方法具有相同的名字但参数列表不同。参数列表可以通过以下方式不同：

  1. 参数的数量不同
  2. 参数的类型不同
  3. 参数的顺序不同

方法重载的规则

  1. 方法名必须相同。
  2. 参数列表必须不同（参数的数量、类型或顺序）。
  3. 返回类型可以相同也可以不同，但仅靠返回类型不同不能构成方法重载。
  4. 方法的修饰符可以相同也可以不同。
