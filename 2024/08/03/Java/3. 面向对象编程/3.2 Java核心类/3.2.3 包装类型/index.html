3.2.3 包装类型

Yuwl 2024-08-03 11:06:28
Categories： Java Tags： java

现在我们来学习 Java 中的包装类型。

什么是包装类型？

Java有8种基本数据类型（byte, short, int, long, float, double, boolean, char），每种基本类型都有对应的包装类：

  1. Byte
  2. Short
  3. Integer
  4. Long
  5. Float
  6. Double
  7. Boolean
  8. Character
    这些类位于java.lang包中，它们将基本类型”包装”成对象。

为什么需要包装类型？

  1. 将基本类型转换为对象，以便在需要对象的地方使用（如集合类）。
  2. 提供了一些有用的方法，如类型转换和值比较。
  3. 支持null值，而基本类型不支持。

注意事项
