1. Java中的字符串

StringBuilder 和 StringBuffer

对于频繁修改的字符串，使用 StringBuilder（非线程安全）或 StringBuffer（线程安全）更高效。

2. 字符编码

字符编码是将字符映射到特定的二进制表示的方式。

常见编码

  • ASCII: 7位编码，表示128个字符。
  • ISO-8859-1: 8位编码，表示256个字符。
  • UTF-8: 可变长度编码，兼容ASCII，广泛使用。
  • UTF-16: Java内部使用的编码方式。
